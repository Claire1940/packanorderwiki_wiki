<Section icon={Target} title="Preventing a Pack an Order Wrong Order">

The most common frustration for new players is realizing they have triggered a **pack an order wrong order** notification. This happens when an item is placed in a box that does not match the customer's request or when the quantity exceeds the required amount. Accuracy is paramount because the game penalizes the time spent removing incorrect items or restarting a shipment.

To maintain a 100% accuracy rating, players should utilize the built-in verification tools provided in the warehouse interface.

| Control | Action | Practical Application |
| :--- | :--- | :--- |
| **E / Left Click** | Pick up / Pack | Use this for all primary interactions with items and boxes. |
| **Q** | Drop Item | Press this immediately if you realize you picked up a "wrong order" item. |
| **R** | Cycle Orders | Switch between active tickets to see if a held item fits a different order. |
| **Scroll Wheel** | Cycle Inventory | Select the specific item you want to pack when carrying multiple products. |
| **Right Click / Y** | Inspect Item | Essential for distinguishing between similar-looking toys like Squishies. |

<Callout type="warning" title="Critical Warning">
Never pack an item if you are uncertain of its name. Use the **Inspect** function (Right Click or Y) to see the full item description. Items like "Matcha Dumplings" and "Red Squishies" can look remarkably similar in low-light warehouse corners.
</Callout>

<Section icon={Swords} title="Advanced Progression and Team Roles">

For players looking to dominate the 2026 leaderboards, coordination is key. A disorganized team is prone to making **pack an order wrong order** mistakes because two players might try to pack the same item into the same box, leading to overages and wasted time.

<Steps>
  <Step num="1" title="Define Responsibilities">
    Assign one player as the 'Picker' (high Speed/Reach) and one as the 'Packer' (high Carry Capacity). The Picker brings items to the station, and the Packer verifies and boxes them.
  </Step>
  <Step num="2" title="Synchronize Order Cycling">
    Use the **R** key to communicate which order is being worked on. If the team is split, ensure each player is focused on a different customer ticket.
  </Step>
  <Step num="3" title="Batch Processing">
    Instead of running one item at a time, use maxed Carry Capacity to bring a full set of items. This minimizes the idle time for the Packer.
  </Step>
  <Step num="4" title="Loading Optimization">
    The final player (the 'Loader') should focus on moving completed boxes to the truck and clearing the floor of any "Q-dropped" items to prevent clutter.
  </Step>
</Steps>

| Team Size | Recommended Strategy | Mistake to Avoid |
| :--- | :--- | :--- |
| **2 Players** | One Picker, One Packer/Loader | Both players leaving the station empty. |
| **3 Players** | One Picker, One Packer, One Loader | Duplicate picking of the same item. |
| **4+ Players** | Zone-based picking (Aisle 1, Aisle 2, etc.) | Crowding the packing station. |

<Callout type="tip" title="Leaderboard Secret">
The fastest teams don't just move fast; they never have to use the **Q** key. Every item picked is an item packed. If you find yourself dropping items frequently, slow down and focus on your **Inspect** mechanic.
</Callout>
