### The Core Packing Loop

Every shift in the warehouse follows a structured sequence. Memorizing these phases ensures you do not fetch the wrong toy or drop valuable cargo.

| Phase | Action | Key Objective |
| :--- | :--- | :--- |
| **1. Review** | Read Ticket | Check product names and exact quantities. |
| **2. Route** | Plan Path | Locate items on shelves before moving. |
| **3. Collect** | Grab Toys | Fill your inventory using optimal reach. |
| **4. Verify** | Inspect Item | Use Right-Click/Y to confirm similar toys. |
| **5. Pack** | Fill Box | Deposit items into the corresponding box. |
| **6. Ship** | Load Truck | Deliver the completed box to secure Coins. |

<Callout type="tip" title="Pro Tip">
Never run to the shelves without checking the active ticket first. Pressing R allows you to cycle through orders so you can plan a route that collects toys for multiple boxes in a single trip.

Navigating the warehouse floor requires precise inputs, especially when carrying multiple items. Misclicking can cause you to drop a toy or place the wrong product into a customer's box, resetting your progress on that checklist.

### Official Control Layout

| Input | Action | Practical Gameplay Tip |
| :--- | :--- | :--- |
| **E / Left Click** | Pick up / Pack | Check the active ticket before interacting to avoid packing errors. |
| **Q** | Drop held item | Drop wrong items near their shelves instead of carrying them to the box. |
| **R** | Cycle active orders | Select your order before entering a shelf zone to maximize efficiency. |
| **Scroll Wheel** | Cycle held items | Switch to the exact product required before approaching the box. |
| **Right Click / Y** | Inspect item | Use this to differentiate items with highly similar shapes or colors. |

### Team Role Distribution

For a three-player team, assign these roles before starting the timer:

1. **The Picker**: Stays in the shelf aisles. Grabs toys listed on the active tickets and drops them at the edge of the packing zone.
2. **The Packer**: Stands at the packing tables. Inspects incoming items, cycles through tickets, and packs them into the correct boxes.
3. **The Loader**: Grabs completed boxes from the tables, runs them to the delivery trucks, and monitors the next incoming order queue.
