<Section icon={Target} title="Mastering the Controls and Item Verification">

In the **pack an order game**, speed means nothing if you pack incorrect items. A single wrong item can ruin your completion time. Utilizing the built-in inspection controls guarantees a 100% accuracy rate.

| Input Key | Action | Practical Warehouse Application |
|-----------|--------|---------------------------------|
| **E / Left Click** | Pick up / Pack | Primary interaction key for shelves and boxes. |
| **Q** | Drop Item | Use this to discard wrong items near their original shelves. |
| **R** | Cycle Orders | Switch active tickets to gather items for multiple orders. |
| **Scroll Wheel** | Cycle Inventory | Switch your active held item before approaching the box. |
| **Right Click / Y** | Inspect Item | Zoom in on an item to confirm its name and color. |

<Callout type="warning" title="Watch Out for Lookalikes">
Several toys share highly similar shapes but have different names or colors. Always use **Right Click** or **Y** to inspect items if you are uncertain. Packing a wrong item forces you to waste time sorting the box.
</Callout>

</Section>
