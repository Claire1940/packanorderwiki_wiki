<Section icon={Map} title="Como Jogar Pack an Order Game: Fluxo de Trabalho para Iniciantes">

O **jogo pack an order** no Roblox é uma simulação de armazém colaborativa e acelerada onde a eficiência é a chave. Seja jogando sozinho ou em equipe, seu objetivo principal é cumprir os tickets dos clientes com precisão antes que o tempo acabe. Completar pedidos recompensa você com Moedas, que são reinvestidas em upgrades de personagem para completar as próximas rodadas ainda mais rápido.

| Passo | Ação | Controle Chave | Dica de Pro |
|------|--------|-------------|---------|
| 1 | Ler Ticket | Pressione R para Alternar | Agrupe pedidos por proximidade de prateleira |
| 2 | Coletar Itens | E / Clique Esquerdo | Carregue vários itens para economizar viagens |
| 3 | Verificar Brinquedos | Clique Direito / Y | Inspecione formas ou cores semelhantes |
| 4 | Embalar Caixa | E / Clique Esquerdo | Solte itens errados com Q perto das prateleiras |
| 5 | Carregar Caminhão | Arraste para o Caminhão | Mantenha as zonas de carregamento livres de bagunça |

Para ajudar você a começar, mapeamos abaixo todo o processo passo a passo do armazém.

<Steps>
  <Step num="1" title="Selecione Seu Modo de Jogo">
    Escolha entre Solo ou Multijogador. Jogar com amigos é altamente recomendado, pois vocês podem dividir as seções do armazém para evitar idas e vindas desnecessárias.
  </Step>
  <Step num="2" title="Analise o Ticket do Cliente">
    Verifique o ticket de pedido ativo na sua tela. Ele lista o nome do cliente, o número do pedido, os itens necessários e as quantidades exatas. Pressione **R** para alternar entre os pedidos disponíveis e encontrar a rota mais eficiente.
  </Step>
  <Step num="3" title="Localize e Recupere os Itens">
    Navegue pelas prateleiras do armazém para encontrar os brinquedos solicitados. Memorizar o layout geral do armazém é a maneira mais rápida de diminuir seus tempos de execução.
  </Step>
  <Step num="4" title="Embale e Entregue">
    Leve os itens coletados para a estação de embalagem. Pressione **E** para colocá-los na caixa. Assim que a lista de verificação estiver completa, pegue a caixa selada e carregue-a no caminhão de entrega para resgatar suas Moedas.
  </Step>
</Steps>

<Section icon={Target} title="Dominando os Controles e Verificação de Itens">

No **jogo pack an order**, velocidade não significa nada se você embalar os itens incorretos. Um único item errado pode arruinar seu tempo de conclusão. Utilizar os controles de inspeção integrados garante uma taxa de precisão de 100%.

| Tecla de Entrada | Ação | Aplicação Prática no Armazém |
|-----------|--------|---------------------------------|
| **E / Clique Esquerdo** | Pegar / Embalar | Tecla de interação primária para prateleiras e caixas. |
| **Q** | Soltar Item | Use isso para descartar itens errados perto de suas prateleiras originais. |
| **R** | Alternar Pedidos | Troque os tickets ativos para reunir itens de vários pedidos. |
| **Roda do Mouse** | Alternar Inventário | Alterne seu item ativo antes de se aproximar da caixa. |
| **Clique Direito / Y** | Inspecionar Item | Aumente o zoom em um item para confirmar seu nome e cor. |

<Callout type="warning" title="Cuidado com Itens Parecidos">
Vários brinquedos compartilham formas muito semelhantes, mas têm nomes ou cores diferentes. Sempre use o **Clique Direito** ou **Y** para inspecionar itens se não tiver certeza. Embalar um item errado força você a perder tempo organizando a caixa.
</Callout>

</Section>

| Função | Responsabilidade Primária | Foco de Upgrade Ideal | Ações Chave |
|------|------------------------|--------------------|-------------|
| **Coletor** | Recuperar itens das prateleiras | Velocidade, Capacidade | Deixa os itens perto das estações de embalagem |
| **Embalador** | Verificar tickets e encher caixas | Alcance, Velocidade | Alterna pedidos, gerencia o inventário |
| **Carregador** | Mover caixas prontas para os caminhões | Velocidade | Mantém a doca de carregamento livre |
