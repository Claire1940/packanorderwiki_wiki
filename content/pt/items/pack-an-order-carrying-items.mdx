### O Ciclo Principal de Embalagem

Cada turno no armazém segue uma sequência estruturada. Memorizar essas fases garante que você não busque o brinquedo errado nem perca cargas valiosas.

| Fase | Ação | Objetivo Principal |
| :--- | :--- | :--- |
| **1. Revisar** | Ler Pedido | Verificar nomes dos produtos e quantidades exatas. |
| **2. Rota** | Planejar Caminho | Localizar itens nas prateleiras antes de se mover. |
| **3. Coletar** | Pegar Brinquedos | Encher seu inventário usando o alcance ideal. |
| **4. Verificar** | Inspecionar Item | Usar o Botão Direito/Y para confirmar brinquedos semelhantes. |
| **5. Embalar** | Encher Caixa | Depositar itens na caixa correspondente. |
| **6. Enviar** | Carregar Caminhão | Entregar a caixa concluída para garantir Moedas. |

<Callout type="tip" title="Dica de Mestre">
Nunca corra para as prateleiras sem verificar o pedido ativo primeiro. Pressionar R permite que você alterne entre os pedidos para planejar uma rota que colete brinquedos para várias caixas em uma única viagem.
</Callout>

Navegar pelo chão do armazém exige comandos precisos, especialmente ao carregar vários itens. Um clique errado pode fazer você derrubar um brinquedo ou colocar o produto errado na caixa de um cliente, reiniciando seu progresso naquela lista de tarefas.

### Esquema de Controles Oficial

| Comando | Ação | Dica Prática de Jogo |
| :--- | :--- | :--- |
| **E / Clique Esquerdo** | Pegar / Embalar | Verifique o pedido ativo antes de interagir para evitar erros de embalagem. |
| **Q** | Soltar item segurado | Solte itens errados perto de suas prateleiras em vez de carregá-los até a caixa. |
| **R** | Alternar pedidos ativos | Selecione seu pedido antes de entrar em uma zona de prateleira para maximizar a eficiência. |
| **Roda do Mouse** | Alternar itens segurados | Mude para o produto exato necessário antes de se aproximar da caixa. |
| **Botão Direito / Y** | Inspecionar item | Use isso para diferenciar itens com formas ou cores muito semelhantes. |

### Prioridade de Melhorias Principais

Existem três melhorias principais disponíveis no armazém. Atualizá-las na sequência correta evita gargalos de eficiência.

<Steps>
  <Step num="1" title="Melhoria de Velocidade">
    Este é o seu atributo mais importante. Ele reduz o tempo de viagem entre prateleiras distantes, as mesas de embalagem e os caminhões de entrega. Compre este primeiro.
  </Step>
  <Step num="2" title="Capacidade de Carga">
    Aumenta o número de itens que você pode carregar simultaneamente. Isso é altamente crítico para jogadores solo que não podem contar com companheiros de equipe para buscar partes de um pedido.
  </Step>
  <Step num="3" title="Melhoria de Alcance">
    Estende a distância a partir da qual você pode pegar brinquedos ou interagir com caixas. Compre por último, assim que sua velocidade de movimento estiver otimizada.
  </Step>
</Steps>

<Section icon={Trophy} title="Estratégias Cooperativas e de Classificação">

Embora jogar solo seja uma ótima maneira de aprender o layout, as corridas de alto nível na classificação exigem equipes multijogador coordenadas. A divisão de tarefas evita que os jogadores corram para a mesma prateleira e dupliquem o trabalho.

### Distribuição de Funções na Equipe

Para uma equipe de três jogadores, atribua estas funções antes de iniciar o cronômetro:

1. **O Coletor**: Fica nos corredores das prateleiras. Pega os brinquedos listados nos pedidos ativos e os solta na borda da zona de embalagem.
2. **O Embalador**: Fica nas mesas de embalagem. Inspeciona os itens que chegam, alterna entre os pedidos e os embala nas caixas corretas.
3. **O Carregador**: Pega as caixas concluídas das mesas, corre com elas para os caminhões de entrega e monitora a próxima fila de pedidos.
