Depois de entender **como ler pedidos no Pack an Order**, você precisa executar as tarefas físicas usando o esquema de controles do jogo.   <Card title="Interações Básicas">
    - **E ou Clique Esquerdo**: Ação primária para coletar produtos e embalá-los nas caixas.
    - **Tecla Q**: Solta o item que está segurando no momento — use isso para liberar espaço se pegar o brinquedo errado.

  <Card title="Navegação de Pedidos">
    - **Tecla R**: Alterna entre os tíquetes de clientes ativos.

  </Card>
  <Card title="Ferramentas de Inspeção">

A tecla \"R\" é sem dúvida o botão mais importante para jogadores avançados. Ao alternar entre os pedidos enquanto está nas prateleiras, você pode realizar \"multitarefas\". Se você estiver perto da prateleira de \"Matcha Dumplings\" para o Pedido nº 1, mas o Pedido nº 2 também precisar deles, você pode coletar os itens para ambos simultaneamente.

| Comando | Ação | Dica Prática |
| :--- | :--- | :--- |
| **E / Clique Esquerdo** | Coletar / Embalar | Posicione-se diretamente em frente à caixa para evitar errar o clique. |
| **Q** | Soltar Item | Solte itens incorretos perto de suas prateleiras originais para ajudar os colegas de equipe. |
| **R** | Alternar Pedidos | Verifique todos os pedidos antes de sair da estação de embalagem. |
| **Roda do Mouse** | Alternar Item Segurado | Sempre selecione o item que deseja embalar antes de clicar. |
| **Clique Direito / Y** | Inspecionar Item | Essencial para distinguir entre cores/formas semelhantes. |

<Callout type="warning" title="Evite Erros de Embalagem">
Se você embalar um item incorreto, não poderá simplesmente retirá-lo. Sempre verifique o nome do item usando a ferramenta de inspeção se não tiver certeza.
</Callout>

<Section icon={Sparkles} title="Fluxo de Trabalho de Embalagem Passo a Passo">

Para dominar o ciclo de jogabilidade de **como ler pedidos no Pack an Order**, você deve seguir um fluxo de trabalho padronizado. Isso reduz a carga mental e evita erros comuns, como correr de um lado para o outro até a mesma prateleira várias vezes.

<Steps>
  <Step num="1" title="Revisar Tíquetes Ativos">
    Fique na estação de embalagem e pressione **R** para olhar todos os pedidos atuais. Memorize os itens necessários e suas localizações gerais no depósito.
  </Step>
  <Step num="2" title="Planejar sua Rota">
    Agrupe os itens pelas localizações das prateleiras. Se o \"Clicker\" e o \"Red Squishy\" estiverem no mesmo corredor, planeje coletar ambos em uma única viagem.
  </Step>
  <Step num="3" title="Coletar e Verificar">
    Navegue até as prateleiras. Colete os itens usando **E**. Se não tiver certeza, use a ferramenta de inspeção com o **Clique Direito** para confirmar se o nome do produto corresponde ao tíquete.
  </Step>
  <Step num="4" title="Embalar e Finalizar">
    Retorne à estação de embalagem. Use a **Roda do Mouse** para selecionar o item correto e pressione **E** para colocá-lo na caixa. Assim que a lista de verificação estiver concluída, carregue-a no caminhão.
  </Step>
</Steps>

| Fase do Fluxo de Trabalho | Objetivo Principal | Ação Recomendada |
| :--- | :--- | :--- |
| **Preparação** | Minimizar deslocamento | Alterne por todos os pedidos antes de se mover. |
| **Coleta** | Maximizar inventário | Preencha sua Capacidade de Carga com itens relevantes. |
| **Verificação** | Zero erros | Use a ferramenta de Inspeção para cada novo tipo de item. |
| **Entrega** | Retorno rápido | Carregue o caminhão imediatamente após embalar o último item. |

<Callout type="success" title="Aumento de Eficiência">
Jogadores solo se beneficiam significativamente ao carregar vários itens. Se você tiver uma Capacidade de Carga de 3, poderá terminar um pedido pequeno em uma única viagem!

Ao jogar com amigos, **como ler pedidos no Pack an Order** se torna um esforço colaborativo. Em vez de todos correrem para os mesmos itens, vocês devem dividir o depósito em funções específicas. Isso evita aglomerações e garante que a estação de embalagem nunca fique ociosa.

| Função | Responsabilidade Principal | Melhores Melhorias |
| :--- | :--- | :--- |
| **O Coletor** | Encontra itens nas prateleiras e os traz para o centro. | Velocidade, Alcance |
| **O Embalador** | Fica na estação, lê os tíquetes e enche as caixas. | Alcance, Capacidade de Carga |

A comunicação eficaz é fundamental. O Embalador deve anunciar quais itens são necessários a seguir, enquanto os Coletores se concentram em navegar pelos corredores. Se o Embalador perceber que um pedido de \"Matcha Dumplings\" está quase pronto, ele pode alertar o Carregador para se preparar para a entrega no caminhão.
