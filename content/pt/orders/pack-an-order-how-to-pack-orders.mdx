<Section icon={Map} title="Como Embalar Pedidos: O Básico da Jogabilidade">

No mundo acelerado do armazém de Roblox, aprender a **embalar pedidos de forma eficiente no Pack an Order** é a chave para subir nas tabelas de classificação e maximizar seus ganhos de Moedas. O jogo desafia você a navegar por prateleiras movimentadas, verificar os nomes dos itens, embalar caixas corretamente e carregar as remessas concluídas nos caminhões de entrega sob limites de tempo apertados.

Seja jogando sozinho ou colaborando com amigos, cada remessa bem-sucedida depende de um fluxo de trabalho estruturado. Gerenciar seu tempo e minimizar viagens desnecessárias pelo armazém impactará diretamente no seu sucesso.

| Etapa | Objetivo Principal | Controles Chave |
| :--- | :--- | :--- |
| **1. Ler** | Verificar os requisitos do tíquete de cliente ativo | **R** para alternar pedidos |
| **2. Coletar** | Recuperar itens correspondentes das prateleiras | **E** ou **Clique Esquerdo** |
| **3. Verificar** | Confirmar se o nome do item e a quantidade correspondem | **Botão Direito** ou **Y** |
| **4. Embalar** | Colocar os itens na caixa de pedido correta | **E** ou **Clique Esquerdo** |
| **5. Entregar** | Carregar a caixa finalizada no caminhão | **E** ou **Clique Esquerdo** |

<Callout type="tip" title="Dica de Mestre">
Sempre alterne entre seus tíquetes ativos antes de sair da estação de embalagem. Planejar uma rota que colete itens para vários pedidos de uma só vez economiza segundos valiosos.
</Callout>

</Section>

<Section icon={Target} title="Guia de Embalagem de Pedidos Passo a Passo">

Para estabelecer uma rotina altamente eficiente, siga este passo a passo de uma rodada padrão no armazém. Dominar este ciclo é essencial para concluir as remessas antes que o tempo acabe.

<Steps>
  <Step num="1" title="Selecione seu Modo de Jogo">
    Escolha entrar no armazém sozinho ou junte-se a um lobby com amigos. O jogo cooperativo permite que as equipes dividam as seções de prateleiras, enquanto o jogo solo exige uma rota individual altamente otimizada.
  </Step>
  <Step num="2" title="Analise o Tíquete do Cliente">
    Olhe para o tíquete de pedido ativo na sua tela. Observe o nome do cliente, o número do pedido, os itens necessários e as quantidades exatas. Por exemplo, Becca pode solicitar 1 Red Squishy, 2 Matcha Dumplings e 3 Salted Butters.
  </Step>
  <Step num="3" title="Navegue pelas Prateleiras e Inspecione os Itens">
    Siga para os corredores do armazém. Se os itens parecerem semelhantes em formato ou cor, segure o **Botão Direito** ou pressione **Y** para inspecioná-los. Isso garante que você não pegue o brinquedo errado.
  </Step>
  <Step num="4" title="Colete e Gerencie seu Inventário">
    Pressione **E** ou **Clique Esquerdo** para pegar o item. Se você tiver melhorado sua capacidade de carga, use a **Roda do Mouse (Scroll)** para alternar entre os itens que está carregando no momento.
  </Step>
  <Step num="5" title="Embale a Caixa e Carregue o Caminhão">
    Retorne à estação de embalagem e pressione **E** para colocar os itens corretos na caixa. Assim que a lista de verificação estiver concluída, pegue a caixa selada e carregue-a no caminhão de entrega para ganhar suas Moedas.
  </Step>
</Steps>

<Callout type="warning" title="Cuidado com seu Inventário">
Se você acidentalmente pegar o item errado, pressione **Q** para soltá-lo imediatamente. Não perca tempo levando produtos incorretos de volta para a estação de embalagem.
</Callout>

</Section>

<Section icon={Shield} title="Referência de Controles e Teclas de Atalho">

Familiarizar-se com o esquema de controles é vital para um movimento fluido e embalagem rápida. Memorize estas teclas de atalho para minimizar erros durante turnos intensos e cronometrados.

| Comando | Ação | Aplicação Prática |
| :--- | :--- | :--- |
| **E / Clique Esquerdo** | Pegar / Embalar item | Interagir com prateleiras, caixas e caminhões |
| **Q** | Soltar item carregado | Descartar itens incorretos perto de suas prateleiras |
| **R** | Alternar pedidos ativos | Revisar diferentes tíquetes de clientes rapidamente |
| **Roda do Mouse (Scroll)** | Alternar itens carregados | Escolher qual item embalar primeiro ao carregar muitos |
| **Botão Direito / Y** | Inspecionar item | Confirmar nomes de brinquedos parecidos antes de pegar |

<Callout type="info" title="Compatibilidade de Dispositivos">
Esses controles são otimizados para jogadores de PC. Jogadores de console e dispositivos móveis encontrarão botões interativos correspondentes em suas telas para realizar as mesmas ações.
</Callout>

</Section>
