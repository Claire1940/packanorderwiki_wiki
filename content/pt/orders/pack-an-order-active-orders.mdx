<Section icon={Map} title="Fluxo de Trabalho Passo a Passo no Armazém">

O sucesso no armazém depende de um fluxo de trabalho lógico e repetível. Os iniciantes costumam sofrer com a síndrome do "vai e vem", onde correm para uma prateleira para pegar um item, voltam para embalá-lo e só então percebem que precisam de outro item da mesma prateleira. Seguir um processo estruturado reduz o tempo ocioso e aumenta o seu rendimento de Moedas por hora.

<Steps>
  <Step num="1" title="Analise o Tíquete Ativo">
    Abra seu pedido atual e memorize os itens. Procure por palavras-chave específicas como "Clicker" ou "Salted Butter". Se tiver vários pedidos, alterne entre eles para encontrar localizações de prateleiras que se sobreponham.
  </Step>
  <Step num="2" title="Planeje a Rota das Prateleiras">
    Mova-se em direção às seções de prateleiras que contêm a maioria dos itens solicitados. Use seu conhecimento do layout do armazém para fazer o caminho mais curto, evitando obstáculos e outros jogadores.
  </Step>
  <Step num="3" title="Verifique e Colete">
    Aproxime-se da prateleira e use a tecla de interação (E ou Clique Esquerdo) para pegar o item. Se dois itens parecerem semelhantes, use a função Inspecionar (Clique Direito ou Y) para confirmar se o nome do produto corresponde ao tíquete.
  </Step>
  <Step num="4" title="Embale e Finalize">
    Retorne à área de embalagem. Use a roda de rolagem do mouse (scroll) para selecionar o item correto do seu inventário e interaja com a caixa para embalá-lo. Assim que a lista de verificação estiver verde, leve a caixa para o caminhão de entrega.
  </Step>
</Steps>

<Callout type="warning" title="Evite Erros de Embalagem">
Embalar um item incorreto não apenas desperdiçará o item, mas também pode exigir que você reinicie a caixa ou remova o item, prejudicando significativamente seu tempo de conclusão. Sempre inspecione brinquedos duvidosos.
</Callout>

| Estágio do Fluxo | Ação Primária | Erro Comum |
| :--- | :--- | :--- |
| **Coleta** | Coletar brinquedos das prateleiras | Pegar itens sem verificar o pedido |
| **Verificação** | Inspecionar nomes dos itens | Basear-se apenas na cor ou formato do brinquedo |
| **Inventário** | Gerenciar itens carregados | Esquecer de rolar para o item correto |
| **Carregamento** | Mover caixas para o caminhão | Deixar caixas concluídas na estação de embalagem |

</Section>

<Section icon={Swords} title="Estratégias Co-op vs Solo">

O jogo suporta tanto o modo solo quanto equipes colaborativas de armazém. Dependendo do tamanho do seu grupo, sua abordagem aos **pedidos ativos no Pack an Order** deve mudar. Em uma equipe, a maior ameaça à eficiência é a duplicidade — onde dois jogadores buscam o mesmo item porque não estão se comunicando sobre qual tíquete estão trabalhando.

**Estratégia Solo:**
No modo solo, você é o coletor, o embalador e o carregador. Seu foco deve ser o gerenciamento de inventário. Como você não tem ninguém para ajudar, use a tecla R constantemente para garantir que nunca esteja andando com o inventário vazio. Se você está indo para o fundo do armazém para o Pedido A, verifique se o Pedido B precisa de algo daquela área.

**Estratégia Multijogador:**
As equipes devem atribuir funções específicas para evitar o caos. Um "Embalador" dedicado fica perto das caixas e caminhões, enquanto os "Coletores" correm para as prateleiras.

<Section icon={Shield} title="Referência de Controles e Navegação">

Dominar os controles é essencial para manter uma alta taxa de \"Pedidos Por Hora\" (OPH). A maioria das interações está mapeada para a tecla E ou Clique Esquerdo, mas jogadores avançados fazem uso constante das teclas secundárias para gerenciar seu inventário sem interromper o movimento.

| Comando | Ação | Dica de Mestre |
| :--- | :--- | :--- |
| **E / Clique Esquerdo** | Pegar / Embalar item | Toque duplo para pegar e guardar rapidamente |
| **Q** | Soltar item atual | Solte itens errados perto de suas prateleiras, não nos corredores |
| **R** | Alternar pedidos ativos | Verifique isso toda vez que terminar um item |
| **Roda do Mouse (Scroll)** | Alternar itens segurados | Alterne para o próximo item enquanto corre de volta |
| **Clique Direito / Y** | Inspecionar produto | Use para diferenciar itens como 'Matcha' vs 'Wasabi' |

A navegação adequada também envolve entender as \"zonas\" das prateleiras. O armazém geralmente é organizado por tipo de item (ex: Brinquedos, Eletrônicos, Alimentos). Memorizar qual corredor contém os \"Matcha Dumplings\" economizará mais tempo do que qualquer melhoria de velocidade jamais faria.

<Callout type="tip" title="Dica de Navegação">
Não corra apenas em linhas retas. Cortar caminho pelas pontas das prateleiras e usar o espaço aberto perto das docas de carregamento pode poupar segundos valiosos do seu tempo total de entrega.
</Callout>
