A base do seu sucesso em Pack an Order reside nas três melhorias principais. Cada um desses atributos aborda um gargalo específico no fluxo de trabalho do armazém. Entender quando melhorar cada atributo pode economizar horas de trabalho ineficiente.

<CardGrid cols={3}>
  <Card title="Melhoria de Velocidade">
    - **Efeito Principal**: Aumenta a velocidade de caminhada pelo armazém.
    - **Melhor Para**: Reduzir o tempo de deslocamento para itens de prateleiras distantes.
    - **Prioridade**: A mais alta para todos os estilos de jogo.
  </Card>
  <Card title="Melhoria de Alcance">
    - **Efeito Principal**: Estende a distância para coletar e embalar.
    - **Melhor Para**: Coleta em ritmo acelerado sem necessidade de posicionamento preciso.
    - **Prioridade**: Alta para \"Embaladores\" e \"Carregadores\" no modo cooperativo.
  </Card>
  <Card title="Capacidade de Carga">
    - **Efeito Principal**: Aumenta o número de itens segurados de uma só vez.
    - **Melhor Para**: Concluir pedidos grandes em uma única viagem.
    - **Prioridade**: Alta para jogadores solo; menor para equipes especializadas.
  </Card>
</CardGrid>

Além dos atributos básicos, o sistema de **habilidades de Pack an Order** introduz vantagens especializadas que podem alterar significativamente sua estratégia. Essas habilidades são projetadas para lidar com as partes mais demoradas do trabalho: procurar por itens obscuros e gerenciar caminhões de entrega de alto volume.

<Steps>
  <Step num="1" title="Analise seu Gargalo">
    Identifique onde você perde mais tempo. Se você tem dificuldade para encontrar itens, procure por habilidades de busca. Se você passa muito tempo caminhando, concentre-se em vantagens de movimento.
  </Step>
  <Step num="2" title="Desbloqueie Slots de Habilidade">
    À medida que sobe de nível, você desbloqueará slots para equipar diferentes habilidades. Comece com vantagens passivas que não exigem ativação manual para manter seu foco nos pedidos.
  </Step>
  <Step num="3" title="Sincronize com os Papéis da Equipe">
    No modo multijogador, garanta que suas habilidades complementem as de seus companheiros de equipe. Um \"Buscador\" deve ter habilidades de destaque de itens, enquanto um \"Carregador\" deve ter bônus de estamina ou velocidade.
  </Step>
  <Step num="4" title="Teste no Modo Solo">
    Antes de tentar uma corrida de classificação com uma nova habilidade, teste seu alcance efetivo e tempo de recarga em uma sessão solo para entender suas limitações.
  </Step>
</Steps>

<Section icon={Map} title="Dominando os Controles do Armazém para Maior Eficiência">

Habilidades e melhorias são tão boas quanto o jogador que as controla. Dominar os controles do PC ou celular é vital para garantir que o aumento de Velocidade ou Alcance não resulte em quedas acidentais de itens ou embalagens incorretas.

| Comando de Controle | Ação Realizada | Dica Profissional |
| :--- | :--- | :--- |
| **E / Clique Esquerdo** | Coletar ou Embalar item | Toque rapidamente quando estiver perto da caixa para minimizar o atraso |
| **Q** | Soltar item segurado | Use isso para limpar itens \"lixo\" perto da prateleira |
| **R** | Alternar pedidos ativos | Sempre verifique o próximo pedido enquanto caminha até o caminhão |
| **Roda do Mouse** | Alternar inventário segurado | Combine o item com a nota antes de chegar à caixa |
| **Clique Direito / Y** | Inspecionar item | Use isso para brinquedos de aparência semelhante para evitar penalidades |

<Callout type="info" title="Mecânica de Inspeção">
Não ignore o recurso de **Inspeção**. Alguns brinquedos, como squishies de cores diferentes ou caixas de formatos semelhantes, parecem quase idênticos. Uma inspeção de 1 segundo é mais rápida do que a penalidade de 5 segundos por embalar o item errado.
</Callout>

</Section>

<Section icon={Target} title="Estratégias de Progressão Solo vs. Cooperativo">

Sua escolha de **habilidades de Pack an Order** deve mudar dependendo se você está trabalhando sozinho ou com uma equipe. Um jogador solo deve ser um \"pau para toda obra\", enquanto um jogador de equipe deve se especializar em uma parte específica do fluxo de trabalho do armazém.

| Componente Estratégico | Prioridade do Jogador Solo | Prioridade da Equipe Cooperativa |
| :--- | :--- | :--- |
| **Atributo Principal** | **Velocidade e Carga** | **Velocidade e Alcance** |
| **Foco de Habilidade** | Localização de Itens e Busca | Velocidade de Interação e Logística |
| **Gerenciamento de Pedidos** | Concluir uma nota inteira de cada vez | Dividir uma nota entre 2 a 3 jogadores |
| **Rota de Movimento** | Circular (Prateleira -> Caixa -> Caminhão) | Linear (Zonas de prateleiras atribuídas) |
